

# Tanl
### Tunneling a new lead

> This idea was first discussed at: 20.01.2025

This web-app listens to any new rows added to a google sheet and gets the phone No. and the name of the person
Every line coresponds to a person that filled out a form and sends them a response with whatsapp cloud api

# Commits:
Every commit title must be written using this structure:
> {category} : {title} - {small description}

These are all the categories:
- feat : feature -> api
- fix : bug fixes
- refactor
- perf : performance improvements
- style : code formating
- test : unit test, code testing
- docs : documentation
- config : config file changes
- ui/ux : ui / ux changes for cliend end
- database/db : db updates
! if necessary use () after every category to specify where the change has been made
